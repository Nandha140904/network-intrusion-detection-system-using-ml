About the Project

What is NIDS?

The Network Intrusion Detection System (NIDS) is a major academic project focused on detecting and classifying cyber attacks using machine learning techniques applied to real-time network traffic data.

It leverages the NSL-KDD dataset ‚Äî the gold standard benchmark for intrusion detection research ‚Äî to train ensemble models capable of distinguishing between normal traffic and four types of attacks: DoS, Probe, R2L, and U2R.

The system integrates a full pipeline from packet capture ‚Üí feature extraction ‚Üí ML classification ‚Üí alert generation ‚Üí real-time dashboard visualization, making it a comprehensive end-to-end security monitoring platform.

  • ‚úî Trained with 125,973 network flow records from NSL-KDD
  • ‚úî Ensemble model: Random Forest + XGBoost with majority voting
  • ‚úî Real-time capture using Scapy on live network interfaces
  • ‚úî Flask + SocketIO backend for WebSocket real-time updates
  • ‚úî SMOTE applied to handle class imbalance during training
